#Action Plan
1. Pick the site
2. Speak to the client for ideas
3. Get some inspiration
4. Once ideas have been collated, create a wireframe
5. Present wireframe to client
6. Discuss any issues with layout and design features
   1. If all is okay then go ahead and start to create basic functionality of site in HTML and CSS using Skeleton.css
   2. If all is not good then back to re-designing and improving the idea further
7. Present functionality of the site
8. If all is okay then start to add styling features to the site
9. Upload files to a file sharing software
10. Download Wordpress
11. Sync all the files with MySQL and use on Wordpress
12. Offer the option to use Custom Fields for the client to change information easily
